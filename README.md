# Pi Setup

Setup a Raspberry Pi for W&M Robo Ops

### What does it do?

- update
- upgrade
- install [vim](http://www.vim.org/)
- create a vimrc
- clone the [pi clients repo](https://github.com/wm-robo-ops/pi-clients)
- clone the [servos repo](https://github.com/wm-robo-ops/servos)

### Usage

```bash
./setup
```
